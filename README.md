# 爬虫分布式协调系统
分布式协调N台服务器执行抓取任务

# 实现原理
爬虫抓取机器注册到zookeeper指定节点，该系统获取节点列表，再把抓取任务均衡在各节点上

# 具备功能
* 抓取任务文件，根据机器数目，实现负载均衡
* 可横向拓展爬虫机器，抓取任务实时均衡
* 机器故障，抓取任务自动转移，重新分布到可用机器上
* 根据指定标识文件，定期更新抓取文件

# 后期功能
各爬虫服务器可实时反馈抓取状态，机器状态，任务状态等信息，存放在节点数据上，根据数据可做相关的监控报表，报警机制等等
